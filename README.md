# To do list

A basic to do list to help you stay organized and track your tasks or habits.

**How to use it:**

1- Create a new task

2- Once completed, you ✔️ of your list

3 - Delete it

**Techonolgies used:**

Created with React App, CSS, Ant Design Icons, UUID

**App run:**

npm start
